## Installation
- Clone the repo (or download it)
- Extract.
- Use Terminal/Console/Powershell, run ```npm install```
- Edit line #3 on ```geocode.js``` and ```weather.js``` and use your own keys.
- Done.

## Usage
- Use ```node app.js -h``` for all the commands & how to use them
